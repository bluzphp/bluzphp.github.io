Bluz\Db\Db::fetchObject

Returns an object containing first row from the result set

Example of usage <code> // Fetch object to stdClass $stdClass = $db->fetchObject('SELECT * FROM some_table WHERE id = ?', array($id)); // Fetch object to new Some object $someClass = $db->fetchObject('SELECT * FROM some_table WHERE id = ?', array($id), 'Some'); // Fetch object to exists instance of Some object $someClass = $db->fetchObject('SELECT * FROM some_table WHERE id = ?', array($id), $someClass); </code>
