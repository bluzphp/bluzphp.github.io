Bluz\Db\Db::fetchRow

Returns an array containing first row from the result set

Example of usage <code> $db->fetchRow("SELECT name, email FROM users WHERE id = ". $db->quote($id)); $db->fetchRow("SELECT name, email FROM users WHERE id = ?", [$id]); $db->fetchRow("SELECT name, email FROM users WHERE id = :id", [':id'=>$id]); </code>

    Signature

    public function fetchRow(string $sql, [array $params = ] )

    Parameters

    $sql — string
    SQL query with placeholders
    "SELECT * FROM users WHERE name = :name AND pass = :pass"
    $params — array
    params for query placeholders (optional)
    array (':name' => 'John', ':pass' => '123456')

    Returns

    array
    array ('name' => 'John', 'email' => 'john@smith.com')
