Bluz\Db\Db::fetchRow

Returns an array containing first row from the result set

Example of usage <code> $db->fetchRow("SELECT name, email FROM users WHERE id = ". $db->quote($id)); $db->fetchRow("SELECT name, email FROM users WHERE id = ?", array($id)); $db->fetchRow("SELECT name, email FROM users WHERE id = :id", array(':id'=>$id)); </code>
