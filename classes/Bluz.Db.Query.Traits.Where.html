. AND .. condition

Adds one or more restrictions to the query results, forming a logical conjunction with any previously specified restrictions.

$sb = new SelectBuilder();
$sb
    ->select('u')
    ->from('users', 'u')
    ->where('u.username LIKE ?', '%Smith%')
    ->andWhere('u.is_active = ?', 1);
